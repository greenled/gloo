Funciones

Además de las funciones básicas de cualquier pastebin (guardar texto y asociarlo a una URL a través de una interfaz gráfica), Pegotes agrega las siguientes (por ahora):

  • APIs
    • API Xml
    • API Json
    • API texto plano
  • Interfaz gráfica
    • Adaptable a distintas resoluciones de pantalla: desde móviles hasta PCs
    • Disponible en español, inglés, francés, italiano, alemán, portugués, ruso y esperanto
  • Administración
    • Script para automatizar la tarea de iniciar/detener/reiniciar la aplicación.
    • Fácil de instalar y configurar. Disponible como un paquete .deb para sistemas basados en Debian
